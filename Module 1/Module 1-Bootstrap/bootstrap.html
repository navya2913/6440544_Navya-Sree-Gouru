2.1: Bootstrap Structure and Files

Explanation of downloaded Bootstrap directory structure:

  • `css/` folder: Contains Bootstrap's compiled CSS files (e.g., `bootstrap.css`, `bootstrap.min.css`). These define the styles for all Bootstrap components, grid, utilities, etc. `.min.css` is the minified (compressed) version for production.
  • `js/` folder: Contains Bootstrap's compiled JavaScript files (e.g., `bootstrap.bundle.js`, `bootstrap.bundle.min.js`). These provide the interactive functionalities like carousels, modals, tooltips, dropdowns, etc. `bootstrap.bundle.min.js` includes Popper.js, which is needed for some components.
  • `icons/` folder (if using Bootstrap Icons): If you download Bootstrap Icons separately, they would typically be in a folder like this, containing SVG files or web fonts.
  • `scss/` folder (if downloading source): Contains the Sass (SCSS) source files. This is where you would customize Bootstrap by modifying variables and recompiling it (as in Exercise 19).

19.1 & 19.2: Customization with Sass

To perform these exercises, you need to set up a Node.js environment with npm.

Steps for Sass Customization:

  1. Initialize npm: Open your terminal in your project root and run `npm init -y`.
  2. Install Bootstrap and Sass: `npm install bootstrap@5.3.3 sass`.
  3. Create SCSS files:
    • Create a `scss/` folder in your project root.
    • Inside `scss/`, create `_variables.scss` (for overrides) and `custom-bootstrap.scss`.
  4. `scss/_variables.scss` example: 
    // Override Bootstrap's default variables here
$primary: #6f42c1; // Custom primary color (purple)
$border-radius: 1rem; // Larger border radius
$enable-shadows: true; // Enable shadows globally
// Add more overrides as needed
  5. `scss/custom-bootstrap.scss` example: 
    // Import custom variables first
@import "variables";

// Import Bootstrap's functions, variables, and mixins
@import "../node_modules/bootstrap/scss/functions";
@import "../node_modules/bootstrap/scss/variables";
@import "../node_modules/bootstrap/scss/mixins";
@import "../node_modules/bootstrap/scss/root";

// Import all of Bootstrap's core components
@import "../node_modules/bootstrap/scss/reboot";
@import "../node_modules/bootstrap/scss/type";
@import "../node_modules/bootstrap/scss/images";
@import "../node_modules/bootstrap/scss/containers";
@import "../node_modules/bootstrap/scss/grid";
@import "../node_modules/bootstrap/scss/tables";
@import "../node_modules/bootstrap/scss/forms";
@import "../node_modules/bootstrap/scss/buttons";
// ... import other components you need ...
@import "../node_modules/bootstrap/scss/nav";
@import "../node_modules/bootstrap/scss/navbar";
@import "../node_modules/bootstrap/scss/card";
@import "../node_modules/bootstrap/scss/utilities";
@import "../node_modules/bootstrap/scss/helpers";

// Optionally add your own custom CSS below Bootstrap imports
  6. Add a build script to `package.json`: 
    "scripts": {
    "sass": "sass scss/custom-bootstrap.scss:css/custom-bootstrap.min.css --style compressed"
},
  7. Compile Sass: Run `npm run sass` in your terminal. This will create `css/custom-bootstrap.min.css`.
  8. Link custom CSS: In your `index.html` `<head>`, replace Bootstrap's default CSS link with ``.

If done correctly, the primary button below (and any other primary elements) will reflect your custom primary color (e.g., purple) and elements with `rounded` will have the larger radius.
